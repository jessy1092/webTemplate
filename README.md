webTemplate
=============
A web template for simple frontend. Using gulp to build simple webserver and livereload.

## Structure

- app
    + asserts
        - data
        - imgs
        - scripts
        - styles
    + views
- vendor
    + angular
    + d3js
    + jquery
    + semantic-ui

## Installation

`npm i`

`npm start`

and then open `http://localhost:3000/`
